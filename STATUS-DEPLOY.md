## 🚀 PRÓXIMO PASSO: RENDER (5 minutos)

### 1. Criar conta no Render

Acesse: **https://render.com/**

Clique em **"Get Started"** e escolha **"Sign in with GitHub"**

### 2. Criar Web Service

1. No painel do Render, clique **"New +"**
2. Selecione **"Web Service"**
3. Autorize acesso aos seus repositórios (se solicitado)
4. Encontre: **age-run-controle-peso**
5. Clique em **"Connect"**

### 3. Configuração do Service

Preencha os campos:

**Name:** `age-run-controle-peso`

**Region:** `Oregon (US West)` ou `Frankfurt (Europe)`

**Branch:** `main` ✅

**Runtime:** `Node`

**Build Command:** `npm install`

**Start Command:** `npm start`

**Instance Type:** Selecione **"Free"** ✅

### 4. Variáveis de Ambiente (OPCIONAL)

Role até **"Environment Variables"** e adicione apenas se quiser emails funcionando:

```
EMAIL_USER = seu-email@gmail.com
EMAIL_PASS = sua-senha-de-app-16-caracteres
```

*Se não configurar, códigos de recuperação aparecerão no console (servidor)*

### 5. Deploy!

Clique em **"Create Web Service"** (botão verde no final da página)

Aguarde 3-5 minutos enquanto o Render:
- ✅ Baixa seu código
- ✅ Instala dependências (npm install)
- ✅ Inicia o servidor
- ✅ Gera URL pública

### 6. Sucesso! 🎉

Quando aparecer **"Your service is live 🎉"**, seu site estará no ar!

URL será algo como:
`https://age-run-controle-peso.onrender.com`
