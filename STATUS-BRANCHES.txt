⚡ PRÓXIMO PASSO (5 MINUTOS)
════════════════════════════

1️⃣  Acessar: https://dashboard.render.com

2️⃣  Selecionar serviço: age-run-controle-peso

3️⃣  Settings > Preview Environments

4️⃣  Configurar:
    • Enable Preview Environments: ✅ ON
    • Auto-Deploy: ✅ Yes  
    • Branches to Preview: staging, feature/*

5️⃣  Save Changes

6️⃣  Testar:
    $ git checkout staging
    $ echo "test" > test.txt
    $ git add test.txt
    $ git commit -m "test: preview"
    $ git push origin staging
    
    → Aguardar 2 min
    → URL preview aparecerá no Render Dashboard
