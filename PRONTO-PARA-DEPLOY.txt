□ Código testado localmente (npm start)
□ Backup de produção feito (./scripts/backup-completo.sh)
□ Sem erros no console do navegador
□ Login funcionando
□ Páginas principais carregando:
  □ /login
  □ /home
  □ /pesagem
  □ /ranking
  □ /bioimpedancia (nova)
□ Dados sendo salvos corretamente
□ IMC calculando
□ Bioimpedância salvando (se testou)


🆘 ROLLBACK (Se algo der errado)
═════════════════════════════════

Se após o deploy houver problemas:

OPÇÃO 1: Rollback via Render (Mais rápido)
──────────────────────────────────────────
1. Dashboard > Deploys (menu lateral)
2. Encontrar deploy anterior (que funcionava)
3. Clicar: "Redeploy" no deploy antigo
4. Aguardar ~2 min
✅ Voltou para versão anterior


OPÇÃO 2: Rollback via Git
──────────────────────────
$ git revert HEAD
$ git push origin main
$ Render Dashboard > Manual Deploy > Deploy
✅ Commit de reversão publicado


OPÇÃO 3: Restaurar Backup
──────────────────────────
$ cd backups/backup-completo-latest/
$ node scripts/restore-database.js database/dados-completos.json
✅ Dados restaurados


💡 DICAS IMPORTANTES
════════════════════

1. 🕐 Horário ideal para deploy:
   • Evite horários de pico (meio-dia, fim de tarde)
   • Prefira manhã cedo ou noite
   • Final de semana tem menos usuários

2. 📢 Comunicação:
   • Avise usuários antes (se mudança grande)
   • Grupo do WhatsApp/Telegram
   • "Sistema em atualização às 20h - 5 min de indisponibilidade"

3. 🧪 Teste após deploy:
   • Faça login você mesmo
   • Teste registro de pesagem
   • Verifique se dados aparecem
   • Teste nova página de bioimpedância

4. 👀 Monitore logs:
   • Dashboard > Logs
   • Fique atento a erros
   • Primeiros 10 minutos são críticos


🎯 FEATURES PRONTAS PARA DEPLOY
════════════════════════════════

Quando você fizer deploy, os usuários terão acesso a:

✅ Nova página: Bioimpedância
   • Menu com 4 itens (Home, Pesagem, Ranking, Bioimpedância)
   • Formulário manual para dados de bioimpedância
   • 8 campos: Peso*, % Gordura, % Músculo, % Água, Massa Óssea,
               Metabolismo Basal, Idade Metabólica, Gordura Visceral

✅ Análise Avançada (quando preencher bioimpedância)
   • Interface com 2 abas
   • Aba 1: IMC Tradicional (peso/altura)
   • Aba 2: Análise Avançada
     - IMG (Índice de Massa Gorda)
     - FFMI (Fat-Free Mass Index)
     - Barras de composição corporal
     - Comparativo IMC vs IMG
     - Classificações detalhadas

✅ Campo Altura
   • Novo campo na página de pesagem
   • Salva altura do usuário
   • Necessário para cálculos de IMC

✅ Ranking Melhorado
   • Agora ordena por % de perda de peso
   • Não apenas kg perdidos
   • Mais justo para todos

✅ Melhorias Visuais
   • Cores da paleta do sistema
   • Barra de progresso IMC funcional
   • Cards alinhados
   • Design responsivo
